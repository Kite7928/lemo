## 2.List集合

1. 有序集合(也称为序列)，用户可以精确控制列表中每个元素的插入位置。用户可以通过整数索引访问元素，并搜索列表中的元素

2. 与Set集合不同，列表通常允许重复的元素

   

List集合的特点:

1. 有索引
2. 可以存储重复元素
3. 元素存取有序
